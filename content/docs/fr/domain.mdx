<Callout type="info">
Pour utiliser votre domaine personnalisé, vous devez configurer vos enregistrements DNS afin de le relier à notre serveur.
</Callout>


<Steps>
<Step>

### Accéder à la gestion DNS

Connectez-vous à votre fournisseur de domaine (GoDaddy, Namecheap, OVH, etc.), puis ouvrez la section :

> **Gestion DNS / DNS Management**

</Step>
<Step>

### Étape 2 : Ajouter les enregistrements DNS

Vous devez ajouter **deux enregistrements DNS** obligatoires :

| Type  | Nom (Host) | Valeur         |
|-------|-----------|-----------------|
| A     | @         | 66.29.149.126   |
| CNAME | www       | moneyfusion.net |

### Explications

- **A (@)** → connecte `mondomaine.com` à notre serveur  
- **CNAME (www)** → redirige `www.mondomaine.com` vers le domaine principal  

<Callout type="warning">
Supprimez tous les anciens enregistrements <strong>@</strong> et <strong>www</strong> avant d’ajouter les nouveaux.
</Callout>


</Step>
<Step>

### Attendre la propagation DNS

<Callout type="info">
La propagation DNS peut prendre de quelques minutes à <strong>24 heures</strong> voire même jusqu’à <strong> 48 heures </strong> dans certains cas.
</Callout>


### Vérification externe

👉 https://dnschecker.org

</Step>
<Step>

### Vérifier et ajouter votre domaine

1. Connectez-vous à votre espace **MoneyFusion**
2. Modifier votre boutique
3. Entrez votre domaine (`mondomaine.com`)
4. Enregistez les modifications

</Step>
</Steps>

## Résultat

- ✅ Domaine validé  
- ✅ Connecté automatiquement à votre boutique  

<Callout type="success">
Le système vérifie automatiquement que votre domaine pointe vers <strong>66.29.149.126</strong>.
</Callout>

---

## Problèmes courants

- Mauvaise IP dans l’enregistrement **A**
- **CNAME www** absent ou incorrect
- Conflit avec un ancien enregistrement
- Propagation DNS non terminée
